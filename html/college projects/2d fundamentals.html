2D Fundamentals: Surface/Drawing


Course Description

2D Fundamentals: Surface/Drawing offers an opportunity to discover and research basic principles, language, and concepts inherent in two-dimensional visual systems. It also gives students an opportunity to learn to think critically, analyze, and apply basic principles to design and art projects.


Tools

Adobe Illustrator
Adobe InDesign

Timeline

September 2019 - December 2019


This book complies all the work I have done in this class for the semester.

  • Contrast of black and white dots/lines
  • Composition of symmetry
  • Experimenting with shades of colors
  • Learning about hue and saturation
  • Learning about typefaces by using words as images
